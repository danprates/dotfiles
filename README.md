# Dotfiles

All my dotfiles.

---

## Preparing workspace

```bash
$ mkdir -p ~/workspace

$ cd ~/workspace

$ git clone https://github.com/danprates/dotfiles.git

$ cd dotfiles
```

## Dotbot

To install, just run in the root of the package.

```
$ ./install
```

We use [dotbot](https://github.com/anishathalye/dotbot) to manager.
